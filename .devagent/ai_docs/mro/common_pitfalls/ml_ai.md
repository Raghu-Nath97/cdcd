# Machine Learning & AI Common Pitfalls

Below are examples of common pitfalls Data Scientists encounter throughout the project lifecycle with a focus on machine learning and AI.

- **Overfitting:** Building models that perform well on training data but poorly on new data.
- **Data leakage:** Accidentally using information from the test set during training.
- **Improper validation:** Not using cross-validation or proper train/test splits.
- **Feature selection mistakes:** Including irrelevant or redundant features.
- **Ignoring model interpretability:** Using black-box models without understanding their decisions.
