# PyGentic - Enterprise Agentic Framework

## Overview

PyGentic is P&G's enterprise agentic framework, designed to be secure, reliable, and scalable. This Python-based framework is seamlessly integrated into P&G's AI Factory, facilitating the development of AI Agents—goal-driven software entities that autonomously complete tasks and achieve objectives.

AI Agents within PyGentic are engineered to:
- Receive instructions in natural language
- Formulate dynamic plans to achieve specified goals
- Autonomously employ various tools and techniques to execute tasks

## Why use PyGentic?

PyGentic was designed by P&G GenAI developers for P&G GenAI developers. It simplifies building end-to-end AI Agentic applications, not just standalone AI Agents.

Key advantages:
- **Built-in API Deployment**: Easily deploy AI Agents as APIs for integration with other applications
- **Built-in Agentic Flow Templates**: Pre-defined templates for common use cases
- **Native Integration with AI Factory**: Seamless integration with GenAI Platform, AI Observability, and IPA
- **Built-in Support for AI Apps Chainlit**: Build interactive interfaces with minimal code
- **Native Integration with chatPG**: Implement agents as chatPG Assistants quickly
