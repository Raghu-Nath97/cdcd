# Software Development Common Pitfalls

Below are examples of common pitfalls Data Scientists encounter throughout the project lifecycle with a focus on software development.

- **Poor version control:** Not using git or failing to document code changes.
- **Lack of reproducibility:** Not using environments or failing to document dependencies.
- **Insufficient testing:** Skipping unit tests or validation of data pipelines.
- **Ignoring scalability:** Building solutions that do not scale with data or users.
